# Backend-Express

## Dependencies:
**bcryptjs**\
**cors**\
**dotenv**\
**express**\
**express-validator**\
**jsonwebtoken**\
**mongoose**\
**nodemon**\
**request**

## You must follow the next steps in order to execute the backend:
**1) npm install**
###
**2) npm run dev (the backend will be run in localhost:4000)**
###
**3) You can test it with this collection https://www.getpostman.com/collections/257f1248846aab9694f9**
